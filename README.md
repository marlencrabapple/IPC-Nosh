# NAME

IPC::Nosh - no-shell system commands and subprocess interaction

# SYNOPSIS

    use IPC::Nosh; # run() is exported by default
    my $run = run(\@cmd, %options)

# DESCRIPTION

IPC::Nosh is a easy to use tool to multiplex data to and from external commands.
